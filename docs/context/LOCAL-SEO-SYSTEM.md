## Overview

A 20-prompt AI-powered local SEO system for home services businesses. Uses browser-controlled AI (Claude Code, Cursor, etc.) to automate audits across GBP, website, backlinks, and content. Runs on SEMrush, Ahrefs, GSC, and GBP data.

## The 20 Prompts — Summary

### PART 1: GBP (Prompts 1–8)
1. **GBP category audit** — compare your categories vs top 3 competitors, find missing secondary categories
2. **GBP attributes audit** — identify every relevant attribute competitors have that you're missing
3. **Review velocity audit** — benchmark review count/rate vs competitors, build a 90-day review acquisition plan
4. **Review response audit** — analyze competitor review responses, build templated response scripts
5. **GBP post analysis** — audit competitor post history, build content calendar
6. **GBP services audit** — extract competitor service listings, find gaps in your own
7. **GBP description optimizer** — scrape competitor descriptions, write 3 optimized versions (keyword/conversion/trust)
8. **GBP photo audit** — benchmark photo count/velocity vs competitors, build 8-week upload plan with naming/geotagging

### PART 2: WEBSITE (Prompts 9–13)
9. **Keyword gap audit** — find keywords competitors rank for (pos 1–20) that you don't, filter by local intent + KD <40
10. **Money page audit** — identify pages ranking 4–15 that need one push, pages with high impressions/low CTR
11. **Service + city page builder** — build full location-specific pages for every service/city combo missing from site
12. **Google Search Console analysis** — find page 2 goldmine (pos 11–20, 100+ impressions), build 30-day sprint
13. **Review sentiment analysis** — extract emotional language from competitor reviews, rewrite GBP + homepage copy to match

### PART 3: BACKLINKS + AUTHORITY (Prompts 14–16)
14. **Competitor backlink audit** — find domains linking to all 3 competitors but not you, build 90-day outreach plan
15. **Local citation audit** — check NAP consistency across 15+ directories, fix inconsistencies, add missing listings
16. **Local search intent mapping** — categorize all keywords by buyer journey stage (problem-unaware → ready to hire)

### PART 4: CONTENT + TRACKING (Prompts 17–20)
17. **Content gap analysis** — find questions competitors rank for that you don't, write content briefs for top 20
18. **Entity optimization** — build schema markup, Wikidata presence, knowledge panel triggers
19. **Competitor GBP posting pattern analysis** — forensic analysis of competitor post timing/format, build rival strategy
20. **Monthly SEO performance report** — GSC + GBP + GA4 rolled into one 5-minute monthly read
